
# Prueba Técnica




## Tecnologías
Node.js
Express
React con Typescript
MongoDB
## Run Backend


Navigate to the backend folder.

Install dependencies

```bash
  npm install
```

Create a .env file at the root of your project directory with the following values:




```bash
PORT=8080
```

Start the server

```bash
  npm run dev
```


## Run Frontend

Navigate to the frontend-pokedex-test folder.

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start 
```
